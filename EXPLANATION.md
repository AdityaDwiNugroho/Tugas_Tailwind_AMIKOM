# Penjelasan Class Tailwind CSS - Tugas Navbar

Berikut adalah penjelasan mengenai class-class Tailwind CSS yang digunakan pada navbar proyek ini:

1. **`bg-purple-800`**
   * **Fungsi:** Mengubah warna latar belakang navbar menjadi ungu.

2. **`flex`**
   * **Fungsi:** Mengaktifkan tata letak flexbox agar elemen di dalamnya sejajar secara horizontal.

3. **`justify-between`**
   * **Fungsi:** Mengatur jarak antar-elemen agar logo berada di ujung kiri dan tombol menu/login di ujung kanan.

4. **`items-center`**
   * **Fungsi:** Menyejajarkan posisi elemen secara vertikal di tengah navbar.

5. **`transition-transform`**
   * **Fungsi:** Mengaktifkan efek transisi pada perubahan skala (scale) elemen (pada logo).

6. **`duration-300`**
   * **Fungsi:** Menentukan durasi transisi selama 300 milidetik (0,3 detik) agar efek berjalan mulus.

7. **`hover:scale-190`**
   * **Fungsi:** Memperbesar skala logo menjadi 190% saat kursor diarahkan ke elemen.

8. **`hidden`**
   * **Fungsi:** Menyembunyikan elemen dari tampilan layar secara default.

9. **`sm:flex`**
   * **Fungsi:** Menampilkan elemen sebagai flexbox pada ukuran layar sedang (breakpoint sm) ke atas.

10. **`rounded`**
    * **Fungsi:** Memberikan sudut melengkung (border-radius) pada tombol Login.

11. **`bg-orange-500`**
    * **Fungsi:** Memberikan warna latar belakang oranye pada tombol Login.

12. **`hover:bg-orange-600`**
    * **Fungsi:** Mengubah warna latar belakang tombol Login menjadi oranye lebih gelap saat di-hover.

13. **`hover:underline`**
    * **Fungsi:** Menampilkan garis bawah pada teks link menu saat disentuh kursor (hover).

14. **`hover:text-amber-300`**
    * **Fungsi:** Mengubah warna teks link menu menjadi kuning amber saat di-hover.

15. **`hover:font-bold`**
    * **Fungsi:** Mengubah font teks link menu menjadi tebal (bold) saat di-hover.

16. **`transition-all`**
    * **Fungsi:** Mengaktifkan transisi animasi yang halus untuk semua perubahan properti pada elemen link menu.

---
**Catatan:** Seluruh komponen antarmuka dalam proyek ini diimplementasikan sepenuhnya menggunakan class bawaan Tailwind CSS tanpa menggunakan kode CSS murni (custom CSS) tambahan.
